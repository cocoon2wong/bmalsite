湖北大学邹斌教授来实验室进行讲座

佚名 2018-09-14 11:22:12

        2018年9月14日，湖北大学邹斌教授，来实验室参观交流，并开展了关于《New Incremental Learning Algorithm with  Support Vector Machine》的讲座。讲座介绍了在使用SVM进行增量学习中，利用马尔可夫链进行有目的地采样，既得到了更好的分类精度，又大大减小了学习所用时间。在讲座上，邹斌教授的弟子王德胜则以《New Lagrangian Support Vector Machines Classification Algorithm》为题，进一步介绍了将马尔可夫链应用于拉格朗日支持向量机的思想，在保证精度的情况下，极大地缩减了算法运行的时间。
